Logach frontend

Register

Чтобы зарегистрироваться надо ввести имя, фамилию, никнейм, почту и пароль. После успешной регистрации пользователь сможет войти на сайт.

Login

Для входа на сайт надо ввести указанные в регистрации никнейм и пароль.

Profile

В профиле выводится данные пользователя.

Theory

На странице теории можно ознакомится с материалом, который поможет решить практику.

Practice

Для решения практики нужно заполнить форму правильными ответами. После нажатия кнопки ответы проверяются и пользователю выводится результат.

Table

На сайте есть сравнительная таблица, где каждый видит свои успехи в освоении логики и успехи других.
